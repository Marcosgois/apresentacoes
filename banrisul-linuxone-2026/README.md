# IBM LinuxONE · Banrisul 2026

Apresentação animada em HTML (estilo IBM Plex / Carbon, tema claro) sobre IBM LinuxONE para o Banrisul.

## Conteúdo
- `index.html` — deck animado, autossuficiente (abre em qualquer navegador; setas ←/→ ou espaço navegam, `F` para tela cheia).
- `Banrisul-LinuxONE-2026.pptx` — versão editável em PowerPoint (mesmo conteúdo, sem as animações).

## Estrutura (32 slides)
1. Contexto de mercado — crise de fornecimento de memória (DRAM)
2. Valor do LinuxONE — performance, disponibilidade, sustentabilidade
3. Cloud Foundation — OpenShift híbrido, deployment (LPAR/KVM) e topologias
4. Data Serving — TCO de Oracle/MongoDB e casos reais (incl. Pix)
5. Seção técnica — Telum II, hierarquia de cache, cores + coprocessadores vs x86, core sparing e RAIM (animados)
