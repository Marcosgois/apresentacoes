# Template — IBM LinuxONE

Deck base (HTML animado) para iniciar apresentações de novos clientes — sem referências a cliente específico.

## Como usar
1. Copie a pasta para um novo deck do cliente.
2. Substitua o placeholder **`[Cliente]`** pelo nome do cliente (3 ocorrências: `<title>`, capa e fechamento).
3. Ajuste capa/fechamento e remova/adicione slides de caso conforme o cliente.

Navegação: setas ←/→ ou espaço · `F` tela cheia.
